**SaveTo** is a Safari5 plugin that allows you to save your links on a set of social bookmarking systems

SaveTo is an MIT-licensed, open-source plugin to Safari5.

** INSTALLATION

Download the distributable version at http://github.com/jeveaux/delicious-saveto-extension/downloads
Then, double click :)


** BUGS

If you think you have discovered a bug in SaveTo you should report it! We keep
track of all bugs and features at http://github.com/jeveaux/delicious-saveto-extension/issues. 
Report the bug with as much information to help us reproduce it.
